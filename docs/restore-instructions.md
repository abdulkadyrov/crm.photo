# Инструкция восстановления

1. Откройте Vakha Studio на нужном устройстве.
2. Перейдите в `Настройки -> Импорт / экспорт`.
3. Выберите тип импорта.
4. Укажите ZIP-архив.
5. Проверьте предварительный отчёт и конфликты.
6. Выберите стратегию: обновить, оставить существующее или создать копию.
7. После импорта сравните количество проектов, учеников, медиа и документов.
