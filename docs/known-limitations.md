# Известные ограничения

- Основная бизнес-логика всё ещё находится в `js/core/legacy-app.js`.
- Старый ZIP-формат пока обслуживается legacy-кодом.
- Прямое сохранение в выбранную папку доступно только в браузерах с File System Access API.
- OPFS не показывается пользователю как папка ПК и может использоваться только как внутренний кеш.
- Receiver для прямой передачи телефон -> ПК не реализован.
- Полная виртуализация больших списков ещё не внедрена.
