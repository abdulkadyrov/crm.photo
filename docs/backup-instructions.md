# Инструкция резервного копирования

1. Откройте `Настройки -> Импорт / экспорт`.
2. Нажмите `Экспорт всех данных`.
3. Сохраните ZIP в отдельную папку или на внешний носитель.
4. Не удаляйте локальные данные до проверки архива.
5. Для проверки откройте чистый профиль браузера и выполните импорт.

Автоматическое удаление локальных данных после экспорта запрещено.
